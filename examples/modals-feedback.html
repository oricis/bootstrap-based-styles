Details

The modal container (div) has an id, in this case: "feedback-modal", and two neccessary CSS classes: "modal" and "d-none".

The modal id is used in the functions to open and close the modal, adding / removind the CSS clases "d-block" and "d-none". The function used to close the modal must be related with the buttons used to close the modal: the x button placed in the upper right corner of the modal and additional buttons to close the modal in other use cases.
